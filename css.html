CSS - Cascading Style Sheets.

Using again the example of the newspaper given on the page HTML, the CSS would be the person responsible for editing all the content sent before by the writer. This person will not only edit what he has received but also try and find a nice and clean way for the user to visualize all the content with no hassle. This person has to be aware of people's necessities and make the design work for everyone.

CSS - Will be resposible for all the design in your Webpage from text color to where images and videos are allocated on your webpage.

Here is an example of how CSS code works:



body {

background-color: #e5e5e5;

}

#ftext {margin-top: 0;}

h1 {text-align: center;

margin-bottom: 3%;

margin-top: 5%;

}

p {

font-style: normal;

text-align: justify;

font-family:Arial, Helvetica, sans-serif;

font-size: 110%;

margin-left: 10%;

margin-right: 15%;

padding-top: 25px;

color: rgb(61, 60, 60);

}

This tool works along with the HTML code and gives it life and a fluid design to the page. But you are not allowed to interact with the page yet. For that we will need JavaScript , you can learn more about it on our next page.
