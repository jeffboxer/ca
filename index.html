Client Side Technologies

Client side programming is the same as programming for your browser. Every detail you can see while navigating through the Internet was programmed to look and function like it does.

Therefore we use 3 important tools that will enhance the design , functionality and interaction of your webpage:

HTML - The structure of your webpage.

CSS - The design of the webpage.

JavaScript - The interaction between the user and the website.

In this website you will learn a little bit more about each element and how they are used separately in order to make everything that you see while navigating on the Internet possible.
