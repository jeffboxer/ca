Here is an example of how the structure of a page works:

<!DOCTYPE html>

<html lang="en">

<head>

<meta charset="UTF-8">

<meta http-equiv="X-UA-Compatible" content="IE=edge">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>Document</title>

</head>

<body>

</body>

</html>

As you can see, it is composed of tags that make no sense without being format , and that is when CSS enters. You will learn more about CSS on the next page.
